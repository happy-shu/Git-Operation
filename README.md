### 基本オペレーション動作
READMEを編集して、アップロードする作業下記の流れ
 - 修正する（追記）
 - Gitのステージングに上げる
 - Commitする 

#### Step0. 修正項目
  - 何か修正を入れる。修正A
  - 修正B

#### Step1. ステージングにあげる
`git add README.md`

#### Step2. Commit する -m はコメント
`git commit -m "Opartation Test"`

#### Step3. Push する（アップロード）
`git push origin`
